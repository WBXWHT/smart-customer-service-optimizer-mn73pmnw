# smart-customer-service-optimizer

智能客服知识库优化与问答模拟系统

项目经历：我曾参与“智能客服知识库优化”项目。针对客户原有客服响应慢、知识查找效率低的问题，我协助梳理了业务问答流程，并利用提示工程优化了基于大模型的智能问答模块。通过设计测试用例并收集反馈，我们迭代改进了意图识别与答案生成策略。项目上线后，客服平均问题解决时间缩短了40%，知识检索准确率提升至92%。我还协助记录了相关业务价值数据，并沉淀了提示模板优化的初步经验。

## 运行
```bash
pip install -r requirements.txt
python main.py
```
